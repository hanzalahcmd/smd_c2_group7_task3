# ✅ My Tasks — Flutter

A minimal Flutter todo app with a clean blue Material 3 UI.

---

## ✨ Features

- **Task List** — Displays tasks as cards with strike-through for completed items
- **Checkbox Toggle** — Mark tasks done/undone instantly
- **Add FAB** — Floating button ready to hook up a "new task" dialog

---

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/flutter-my-tasks.git
cd flutter-my-tasks
flutter pub get
flutter run
```

No external packages required.

---

## 🏗️ Structure

```
lib/
└── main.dart
    ├── MyApp           # Entry point, blue Material 3 theme
    └── TodoListPage    # Stateful task list with checkbox toggles
```
